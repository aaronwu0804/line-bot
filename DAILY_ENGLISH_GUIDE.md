# 📚 每日英語功能使用指南

## 功能簡介

每日英語功能為 LINE Bot 提供每日不重複的英語單字學習服務,適合國小六年級到國中二年級的學生使用。

### ✨ 主要特色

- 📅 **365天不重複** - 一年365天,每天都有不同的單字
- 📖 **完整學習內容** - 包含單字、詞性、中文意思、例句和翻譯
- 🔊 **發音功能** - 提供單字發音連結,可以聽正確發音
- 🎯 **適合程度** - 國小六年級到國中二年級單字範圍

## 使用方法

在 LINE 聊天室中輸入以下任一指令:

- `每日單字`
- `每日英語`
- `Daily English`
- `單字`

系統會自動回覆今天的每日單字內容。

## 回覆格式

```
📚 每日英語 Daily English

🔤 單字 Word
energy (n.)

📖 中文 Meaning
活力;能量

✏️ 例句 Example
Exercise gives you more energy.

📝 翻譯 Translation
運動讓你更有活力。

💡 Keep learning! 持續學習!

🔊 點擊連結聽發音:
[語音連結]
```

## 技術說明

### 單字資料庫

- 包含 50 個基礎單字
- 自動擴充至 365 天
- 根據日期自動選擇對應的單字

### 發音功能

- 使用 Google Text-to-Speech API
- 提供線上發音連結
- 支援直接點擊播放

### 整合方式

1. **服務模組**: `src/daily_english_service.py`
2. **主程式整合**: `app.py` 中的 `handle_message` 函數
3. **觸發檢測**: 檢查用戶輸入的關鍵字

## 開發者筆記

### 添加新單字

編輯 `src/daily_english_service.py` 中的 `DAILY_WORDS` 列表:

```python
{
    "word": "單字",
    "pos": "詞性(n./v./adj./adv.)",
    "meaning": "中文意思",
    "sentence": "英文例句",
    "translation": "例句翻譯"
}
```

### 調整單字程度

修改 `DAILY_WORDS` 列表,選擇適合的單字難度範圍。

### 自訂發音服務

修改 `get_word_audio_url()` 函數,可以更換為其他 TTS 服務。

## 常見問題

### Q: 為什麼每天的單字都一樣?

A: 系統根據當天日期(一年中的第幾天)來選擇單字,同一天所有用戶看到的單字都相同。

### Q: 可以查看其他天的單字嗎?

A: 目前只支援查看當天的單字,未來可以加入歷史查詢功能。

### Q: 發音連結無法播放怎麼辦?

A: 請檢查網路連線,或嘗試使用不同的瀏覽器開啟連結。

### Q: 可以自己選擇單字嗎?

A: 目前是系統自動分配,未來可以加入自選單字或主題功能。
