## ✅ 待辦事項

### 查看待辦清單
```

花生 我的待辦
花生 查看待辦
花生 顯示待辦
花生 今天要幹嘛
花生 明天要幹嘛
```

### 創建待辦事項
```
花生 提醒我明天要開會
花生 幫我記得下週要交報告
花生 別忘了今天要買菜
花生 新增待辦：完成專案文件
```

### 標記待辦完成
以下是**正確的格式**：

**方式 1：「完成了 + 待辦內容關鍵字」**
```
花生 完成了開會
花生 開會完成了
```

**方式 2：「待辦關鍵字 + 已完成」**
```
花生 寫報告已完成
花生 買菜做完了
```

**方式 3：「標記完成 + 待辦關鍵字」**
```
花生 標記完成開會
花生 標記完成 Python 學習
```

**❌ 錯誤示範**：
```
花生 完成          # 沒有指定是哪個待辦
花生 待辦完成      # 太模糊
```

**💡 提示**：
- 只需要包含待辦內容的**關鍵字**即可
- 例如待辦是「提醒我明天要開會」，說「完成了開會」就可以
- 系統會自動找到包含該關鍵字的待辦事項

### 範例對話

**情境：創建並完成待辦**

```
     ✅ 已完成：
```


## 📝 常見問題

### Q: 為什麼待辦沒有被標記為完成？
A: 請確保：
1. 訊息中包含「完成」、「做完」等關鍵字
2. 包含待辦內容的關鍵字（不需要完整內容）
3. 格式範例：「花生 開會完成了」、「花生 完成了寫報告」

### Q: 如何查看之前儲存的知識？
A: 使用以下任一指令：
- 「花生 我學了什麼」
- 「花生 查看知識」
- 「花生 我的知識」

### Q: 可以同時查看所有類型的內容嗎？
A: 目前需要分別查詢不同類型（知識、靈感、音樂、生活），未來版本會加入統一查詢功能。

### Q: 待辦事項會自動提醒嗎？
A: 目前版本僅儲存待辦，尚未實作自動提醒功能，需要主動查詢「花生 待辦事項」。

## 🎯 使用技巧

1. **關鍵字盡量明確**：標記完成時，使用待辦中的關鍵詞，避免太模糊
2. **善用查詢功能**：定期查看「我的知識」和「待辦事項」來回顧
3. **分類儲存**：不同類型的內容會自動分類，方便日後查找
4. **自然對話**：不需要記住固定格式，花生會理解您的意圖
